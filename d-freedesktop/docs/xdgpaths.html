    Module xdgpaths

    Getting XDG base directories.

    Note

    These functions are defined only on freedesktop systems.

    See Also

    XDG Base Directory Specification

    Functions

    Name Description
    xdgAllConfigDirs The ordered set of non-empty base paths to search for configuration files, in descending order of preference.
    xdgAllDataDirs The ordered set of non-empty base paths to search for data files, in descending order of preference.
    xdgCacheHome The base directory relative to which user-specific non-essential files should be stored.
    xdgConfigDirs The ordered set of non-empty base paths to search for configuration files, in descending order of preference.
    xdgConfigHome The base directory relative to which user-specific configuration files should be stored.
    xdgDataDirs The ordered set of non-empty base paths to search for data files, in descending order of preference.
    xdgDataHome The base directory relative to which user-specific data files should be stored.
    xdgRuntimeDir Path to runtime user directory.
