  • treeMimeType

    Declaration

    nothrow @safe string treeMimeType(string mountPoint, in string[] mimePaths);

    Detect MIME type of mounted device using treemagic files.

    Parameters

    string mountPoint

    Path to mount point of device.

    string[] mimePaths

    Base mime directories where treemagic files are usually placed.

    Return Value

    MIME type name or null if could not detect.

    See Also

    Examples

    1. auto mimePaths = ["test/mime"]; assert(treeMimeType("test/media/photos", mimePaths) == "x-content/image-dcf"); assert(treeMimeType("test/media/installer", mimePaths) == "x-content/unix-software"); assert(treeMimeType("test", mimePaths) == null);
