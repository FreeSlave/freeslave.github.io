    Enum ParamSupport

    Flag set of parameter kinds supported by application. Having more than one flag means that Exec command is ambiguous.

    The enum base type is int .

    Enum members

    Name Description
    file Application can open single file at time.
    files Application can open multiple files at time.
    none Application does not support parameters.
    url Application understands URL syntax and can open single link at time.
    urls Application supports URL syntax and can open multiple links at time.
