    Module desktopfile.utils

    Utility functions for reading and executing desktop files.

    See Also

    Desktop Entry Specification

    Functions

    Name Description
    desktopId See Desktop File ID
    desktopId See Desktop File ID
    expandExecArgs Expand Exec arguments (usually returned by unquoteExec) replacing field codes with given values, making the array suitable for passing to spawnProcess. Deprecated field codes are ignored.
    findDesktopFile Find desktop file by Desktop File ID. Desktop file ID can be ambiguous when it has hyphen symbol, so this function can try both variants.
    findDesktopFile ditto
    getTerminalCommand Detect command which will run program in terminal emulator.
    isTrusted Check if .desktop file is trusted.
    needMultipleInstances Check if application should be started multiple times to open multiple urls.
    paramSupport Evaluate ParamSupport flags for application Exec command.
    shootDesktopFile Read the desktop file and run application or open link depending on the type of the given desktop file.
    shootDesktopFile ditto, but automatically create IniLikeReader from the file.
    spawnApplication Spawn application with given params.
    unquoteExec Apply unquoting to Exec value making it into an array of escaped arguments. It automatically performs quote-related unescaping.
