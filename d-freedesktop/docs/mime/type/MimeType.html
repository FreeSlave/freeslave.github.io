    Class MimeType

    Represents single MIME type.

    Inherits from

    • Object (base class)

    Constructors

    Name Description
    this Create MIME type with name. Name should be given in the form of media/subtype.

    Methods

    Name Description
    addAlias Add alias for this MIME type.
    addMagic Add magic rule.
    addParent Add parent type for this MIME type.
    addPattern Add glob pattern for this MIME type.
    addTreeMagic Add treemagic rule.
    aliases Aliases to this MIME type.
    clearAliases Remove all aliases.
    clearMagic Remove all magic rules.
    clearParents Remove all parents.
    clearPatterns Remove all glob patterns.
    clearTreeMagic Remove all treemagic rules.
    clone Create MimeType deep copy.
    genericIcon Set generic icon name.
    genericIcon Get generic icon name. Use this if the icon could not be found.
    getGenericIcon Get generic icon name. The difference from genericIcon property is that this function provides default generic icon name if no explicitly set. The default form is media part of MIME type name with '-x-generic' appended.
    getIcon Get icon name. The difference from icon property is that this function provides default icon name if no explicitly set. The default form is MIME type name with '/' replaces with '-'.
    icon Set icon name.
    icon Get icon name.
    magics Magic rules for this MIME type.
    name Set MIME type name.
    name The name of MIME type.
    namespaceUri Set namespace uri.
    namespaceUri Get namespace uri for XML-based types.
    parents First level parents for this MIME type.
    patterns Array of MIME glob patterns applied to this MIME type.
    treeMagics Treemagic rules for this MIME type.
