    Module mime.common

    Common functions and constants to work with MIME types.

    Functions

    Name Description
    defaultGenericIconName Default generic icon for MIME type.
    defaultIconName Default icon for MIME type.
    implicitParent Get implicit parent type if mimeType. This is text/plain for all text/* types and application/octet-stream for all streamable types.
    isNoGlobs Check is pattern is _NOGLOBS__. This means glob patterns from the less preferable MIME paths should be ignored.
    isNoMagic Check if value is _NOMAGIC__. This means magic rules from the less preferable MIME paths should be ignored.
    isValidMimeTypeName Check if name is valid MIME type name.
    parseMimeTypeName Parse MIME type name into pair of media and subtype strings.

    Enum values

    Name Type Description
    defaultGlobWeight Default glob pattern weight to use when it's not explicitly provided.
    defaultMatchWeight Default magic rule priority to use when it's not explicitly provided.
