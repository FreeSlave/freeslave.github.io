      Create with msg, lineNumber and fileName.

    • lineNumber

      Declaration

      const pure nothrow @nogc @safe size_t lineNumber();

      Number of line in the file where the exception occured, starting from 1. 0 means that error is not bound to any existing line, but instead relate to file at whole (e.g. required group or key is missing). Don't confuse with line property of Throwable.

    • lineIndex

      Declaration

      const pure nothrow @nogc @safe size_t lineIndex();

      Number of line in the file where the exception occured, starting from 0. Don't confuse with line property of Throwable.

    • fileName

      Declaration

      const pure nothrow @nogc @safe string fileName();

      Name of ini-like file where error occured. Can be empty if fileName was not given upon creating. Don't confuse with file property of Throwable.

    • entryException

      Declaration

      pure nothrow @nogc @safe IniLikeEntryException entryException();

      Original which caused this error. This will have the same msg.

      Return Value

      object or null if the cause of error was something else.
