    Enum IniLikeFile.DuplicateGroupPolicy

    Behavior on group with duplicate name in the file.

    The enum base type is ubyte .

    Enum members

    Name Description
    preserve Preserve all duplicates in the list. The first found group remains accessible by key.
    skip Skip duplicate without error.
    throwError Throw error on group with duplicate name.
