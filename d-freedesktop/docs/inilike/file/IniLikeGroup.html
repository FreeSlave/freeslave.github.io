    Class IniLikeGroup

    This class represents the group (section) in the ini-like file. Instances of this class can be created only in the context of IniLikeFile or its derivatives.

    Inherits from

    • Object (base class)

    Constructors

    Name Description
    this Create instance on IniLikeGroup and set its name to groupName.

    Methods

    Name Description
    addCommentAfter Add comment after some node.
    addCommentBefore Add comment before some node.
    appendComment Add comment line into the group.
    appendValue Add key-value entry without association of value with key. Can be used to add duplicates.
    byIniLine
    byKeyValue Iterate by Key-Value pairs. Values are left in escaped form.
    byNode Iterate over nodes of internal list.
    contains Tell if group contains value associated with the key.
    emptyByKeyValue Empty range of the same type as byKeyValue. Can be used in derived classes if it's needed to have empty range.
    getNode Get internal list node for key.
    groupName Get name of this group.
    localizedValue Perform locale matching lookup as described in Localized values for keys.
    moveLineAfter Move line after other line in the group.
    moveLineBefore Move line before other line in the group.
    moveLineToBack Move line to the end of group.
    moveLineToFront Move line to the start of group.
    opIndex
    opIndexAssign Assign localized value.
    opIndexAssign Insert new value or replaces the old one if value associated with key already exists.
    prependComment Add comment line at the start of group (after group header, before any key-value pairs).
    readEntry Get value by key. This function automatically unescape the found value before returning.
    removeEntry Removes entry by key. Do nothing if not value associated with key found.
    removeEntry ditto, but remove entry by localized key
    removeEntry ditto, but remove entry by node.
    setLocalizedValue Same as localized version of opIndexAssign, but uses function syntax.
    setValue Set value associated with key.
    value Get value by key.
    writeEntry ditto, localized version
    writeEntry Set value by key. This function automatically escape the value (you should not escape value yourself) when writing it.
    validateKey Validate key before setting value to key for this group and throw exception if not valid. Can be reimplemented in derived classes.
    validateValue Validate value for key before setting value to key for this group and throw exception if not valid. Can be reimplemented in derived classes.
