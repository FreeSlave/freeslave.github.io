    Module icontheme.lookup

    Lookup of icon themes and icons.

    Note

    All found icons are just paths. They are not verified to be valid images.

    See Also

    Icon Theme Specification

    Functions

    Name Description
    findClosestIcon Find icon closest of the size. It uses icon theme cache wherever possible. The first perfect match is used.
    findClosestIcon ditto, but with predefined extensions and fallback allowed.
    findFallbackIcon Find fallback icon outside of icon themes. The first found is returned.
    findIconTheme Find index.theme file by theme name.
    findLargestIcon Find icon of the largest size. It uses icon theme cache wherever possible.
    findLargestIcon ditto, but with predefined extensions and fallback allowed.
    iconSizeDistance Distance between desired size and minimum or maximum size value supported by icon theme subdirectory.
    iconThemePaths Find all icon themes in searchIconDirs.
    lookupFallbackIcon Lookup icon alternatives beyond the icon themes. May be used as fallback lookup, if lookupIcon returned empty range.
    lookupFallbackIcons Iterate over all icons out of icon themes.
    lookupIcon Lookup icon alternatives in icon themes. It uses icon theme cache wherever it's loaded. If searched icon is found in some icon theme all subsequent themes are ignored.
    lookupIconTheme Lookup index.theme files by theme name.
    lookupThemeIcons Iterate over all icons in icon themes. iconThemes is usually the range of the main theme and themes it inherits from.
    matchBestIcon Find icon closest to the given size among given alternatives.
    matchIconSize Check if matchSize belongs to subdir's size range.
    openBaseThemes Recursively find all themes the given theme is inherited from.
    openIconTheme Find index.theme file for given theme and create instance of IconThemeFile. The first found file will be used.

    Structs

    Name Description
    IconSearchResult Result of icon lookup.

    Enum values

    Name Type Description
    defaultIconExtensions Default icon extensions. This array includes .png and .xpm. PNG is recommended format. XPM is kept for backward compatibility.
